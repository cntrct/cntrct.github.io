CntrcT: Contracting Contracts

This is a placemarker website for Cntrct.org. Please see the www.DecentLaw project.

CntrcT and DecentLaw seek to move legal document sourcing, maintenance and collaboration to git and GitHub. To allow lawyers to work with the tools of software coding. To improve the quality and speed of transacting, reduce sophistication barriers, to work at the speed of the web.

DecentLaw seeks to formalize the work of CommonAccord.org.
